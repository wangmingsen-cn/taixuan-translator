## 演示流程

### 第一部分：产品介绍（2分钟）

**话术**：
> 太玄智译是一款面向学术/技术文档的 PDF 精准翻译工具，支持 10+ 主流翻译引擎，可以一键生成符合出版标准的双语对照文档。

**展示要点**：
- 多引擎支持（DeepSeek/OpenAI/Claude/Qwen/Gemini/DeepL/Ollama）
- 出版级 Word 导出（符合 GB/T 7713.1-2006）
- 术语一致性保持
- 本地部署，数据不出网

---

### 第二部分：核心功能演示（12分钟）

#### 2.1 上传与解析（3分钟）

**操作**：
1. 浏览器访问 http://127.0.0.1:8000
2. 点击"选择文件"或拖拽 PDF 到上传区域
3. 观察解析进度（显示段落数、页数）

**解说**：
> 系统会自动识别 PDF 中的段落、表格、公式，保持原有版式。

#### 2.2 引擎选择与翻译（4分钟）

**操作**：
1. 点击"设置"标签
2. 选择翻译引擎（演示用 DeepSeek）
3. 返回主界面，点击"开始翻译"

**解说**：
> 支持多引擎切换，不同引擎有不同特点。DeepSeek 性价比最高，适合大批量翻译。

**展示术语缓存**：
> 翻译过的专业术语会自动缓存，下次遇到相同术语自动使用已有翻译。

#### 2.3 双语对照阅读（3分钟）

**操作**：
1. 翻译完成后，浏览双语对照列表
2. 展示"待翻译/翻译中/已完成/缓存命中"状态
3. 点击任意段落查看详情

**解说**：
> 左边原文，右边译文。状态标记让你清晰了解翻译进度。

#### 2.4 Word 导出（2分钟）

**操作**：
1. 翻译完成后，点击"导出 Word"
2. 选择导出格式（双语对照/仅译文/仅原文）
3. 下载并打开生成的 .docx 文件

**解说**：
> 导出文件符合学术出版规范，格式整洁，可直接投稿。

### 第三部分：技术亮点（3分钟）

**展示要点**：
- **本地部署**：数据不出网，适合企业内部使用
- **多引擎**：按需切换，灵活配置
- **术语库**：自动积累，译名一致
- **缓存机制**：重复内容秒级响应
- **API 接口**：支持二次开发集成

**演示 API 接口**：
1. 访问 http://127.0.0.1:8000/docs
2. 展示 Swagger UI
3. 演示 `/api/engines` 接口

### 第四部分：Q&A（3分钟）

**常见问题**：

| 问题 | 回答 |
|------|------|
| 需要联网吗？ | 需要，翻译 API 需要网络 |
| 支持离线吗？ | 支持 Ollama 本地模型（需较大内存） |
| 支持多少语言？ | 理论上支持所有语言，演示主要中英翻译 |
| 如何收费？ | 按翻译字符数收费（各引擎不同），DeepSeek 最便宜 |
