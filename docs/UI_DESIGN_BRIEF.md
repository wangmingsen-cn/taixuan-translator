## 配色方案（当前实现）

```css
:root {
    /* 主色调 */
    --primary: #2563eb;      /* 蓝色 */
    --primary-dark: #1d4ed8;
    
    /* 状态色 */
    --success: #10b981;     /* 绿色 */
    --warning: #f59e0b;      /* 橙色 */
    --error: #ef4444;       /* 红色 */
    
    /* 背景色（深色主题） */
    --bg-dark: #0f172a;      /* 最深 */
    --bg-card: #1e293b;      /* 卡片 */
    --bg-hover: #334155;     /* 悬停 */
    
    /* 文字色 */
    --text-primary: #f1f5f9; /* 主文字 */
    --text-secondary: #94a3b8;/* 次要文字 */
    
    /* 边框 */
    --border: #334155;
}
```

## 设计要点

### 1. 卡片设计
- 背景: `--bg-card`
- 圆角: `8px`
- 阴影: `0 4px 6px -1px rgba(0,0,0,0.3)`
- 内边距: `16px` 或 `20px`

### 2. 按钮
- 主按钮: `--primary` 背景，白色文字
- 次按钮: `--bg-hover` 背景，`--text-primary` 文字
- 成功按钮: `--success` 背景
- 圆角: `8px`
- 内边距: `8px 16px`

### 3. 输入框
- 背景: `--bg-hover`
- 边框: `1px solid --border`
- 圆角: `8px`
- 聚焦: `border-color: --primary`

### 4. 动画
- 过渡时长: `0.2s` 或 `0.3s`
- 缓动函数: `ease`, `ease-out`
